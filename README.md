# C - Skeleton

## Setup

- Run `` cd <project_dir> ``
- Run `` chmod +x ./bin/console ``

## Requirements

- Run `` cd <project_dir> ``
- Run `` bin/console requirements ``

## Build

- Run `` cd <project_dir> ``
- Run `` bin/console build (debug|release) ``

## Run

- Run `` cd <project_dir> ``
- Run `` bin/console run ``
